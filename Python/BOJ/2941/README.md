# BOJ 2941 - 크로아티아 알파벳

## 문제 설명
- [문제 링크](https://www.acmicpc.net/problem/2941)
- 크로아티아 알파벳은 몇 개의 특수한 문자 조합(예: `c=`, `dz=`, `lj` 등)을 한 글자로 간주한다.
- 주어진 단어가 몇 개의 크로아티아 알파벳으로 이루어져 있는지 계산하는 문제이다.

## 아이디어
- 문자열에서 크로아티아 알파벳 문자열들을 찾아 다른 한 글자의 특수 문자(예: `q`)로 치환한다.
- 치환된 후의 문자열 길이를 측정하면 전체 알파벳 개수를 쉽게 구할 수 있다.

## 시간복잡도
- 치환할 알파벳 리스트의 크기가 상수(8개)이므로, 전체 시간복잡도는 문자열의 길이에 비례하는 `O(L)`이다.

## 풀이 과정
1. 크로아티아 알파벳 리스트 `croatian`을 정의한다. (`dz=`를 가장 먼저 확인해야 한다.)
2. 단어를 입력받는다.
3. 리스트에 있는 각 패턴에 대해 문자열에서 해당 패턴을 한 글자(`q`)로 치환한다.
4. 치환이 완료된 문자열의 길이를 출력한다.

## 코드 설명
```python
import sys
input = sys.stdin.readline

word = input().strip()

croatian = ['dz=', 'c=', 'c-', 'd-', 'lj', 'nj', 's=', 'z=']
for c in croatian :
    word = word.replace(c,'q')
print(len(word))
```
- `replace` 메서드를 사용하여 특정 패턴을 일괄적으로 치환한다.
- `dz=`는 `z=`를 포함하고 있으므로, `dz=`를 먼저 치환해야 의도치 않은 중복 처리를 방지할 수 있다.
- 최종적으로 변환된 `word`의 길이는 문제에서 요구하는 '글자 수'와 일치하게 된다.
