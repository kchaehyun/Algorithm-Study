# BOJ 2841 - 외계인의 기타 연주

## 문제 설명
- [문제 링크](https://www.acmicpc.net/problem/2841)
- 6개의 기타 줄이 있고, 각 줄마다 여러 개의 프렛을 누를 수 있다.
- 어떤 줄의 특정 프렛을 누르려고 할 때, 이미 그 줄에 더 높은 프렛이 눌려 있다면 손가락을 떼어야 한다.
- 이미 같은 프렛을 누르고 있다면 아무 작업도 하지 않는다.
- 손가락을 한 번 누르거나 떼는 것을 '손가락의 움직임' 1회로 간주할 때, 총 움직임의 최솟값을 구하는 문제이다.

## 아이디어
- 각 기타 줄(1~6번)을 독립적인 스택(Stack)으로 관리한다.
- 특정 프렛을 누를 때:
    1. 해당 줄의 스택이 비어있지 않고, 현재 누르려는 프렛보다 높은 프렛이 스택 top에 있다면 `pop`하고 움직임 횟수를 증가시킨다 (손가락을 떼는 행위).
    2. 스택의 top이 현재 프렛과 같다면 이미 누르고 있는 것이므로 넘어간다.
    3. 스택이 비어있거나 top이 현재 프렛보다 낮다면 `push`하고 움직임 횟수를 증가시킨다 (손가락을 누르는 행위).

## 시간복잡도
- 입력의 개수 `N`만큼 반복하며, 각 프렛은 스택에 최대 한 번 들어갔다가(`push`) 한 번 나올(`pop`) 수 있다.
- 따라서 전체 시간복잡도는 `O(N)`이다.

## 풀이 과정
1. 멜로디의 음의 수 `N`과 프렛의 수 `P`를 입력받는다.
2. 6개의 빈 리스트를 담은 `stacks`를 생성한다 (인덱스 편의상 7개 생성).
3. 각 음(줄 번호, 프렛 번호)을 순회한다.
4. `while` 루프를 통해 스택의 마지막 값이 현재 프렛보다 크면 `pop`하고 `ans`를 1 증가시킨다.
5. 루프 종료 후 스택이 비어있지 않고 마지막 값이 현재 프렛과 같다면 추가 동작 없이 다음 음으로 넘어간다.
6. 그렇지 않다면 현재 프렛을 `append`하고 `ans`를 1 증가시킨다.
7. 최종 `ans`를 출력한다.

## 코드 설명
```python
import sys
input = sys.stdin.readline

N, P = map(int, input().split())
stacks = [[] for _ in range(7)]
ans = 0
for i in range(N) :
    line, fret = map(int, input().split())
    while stacks[line] :
        if stacks[line][-1] <= fret :
            break
        stacks[line].pop()
        ans += 1
    if stacks[line] and fret == stacks[line][-1] :
        continue
    stacks[line].append(fret)
    ans += 1

print(ans)
```
- `stacks[line][-1]`을 사용하여 현재 해당 줄에서 가장 높게 눌린 프렛을 확인한다.
- `while`문을 통해 현재 프렛보다 높은 프렛들을 모두 제거함으로써 그리디하게 최소 움직임을 보장한다.
