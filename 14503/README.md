# BOJ 14503 - 로봇 청소기

## 문제 설명
- [문제 링크](https://www.acmicpc.net/problem/14503)
- `N x M` 크기의 방에서 로봇 청소기가 정해진 규칙에 따라 이동하며 청소할 때, 청소하는 칸의 개수를 구하는 시뮬레이션 문제이다.
- 작동 규칙:
  1. 현재 칸이 청소되지 않았다면 청소한다.
  2. 현재 칸 주변 4칸 중 청소되지 않은 빈 칸이 있다면 반시계 방향으로 회전 후 전진한다.
  3. 주변 4칸 중 청소되지 않은 빈 칸이 없다면 바라보는 방향을 유지한 채로 한 칸 후진한다.
  4. 후진할 수 없다면 작동을 멈춘다.

## 아이디어
- 로봇의 위치(`r, c`)와 방향(`d`)을 상태 변수로 관리하며 규칙을 `while True` 루프로 구현한다.
- 방향은 북(0), 동(1), 남(2), 서(3) 순서이며, 반시계 회전은 `(d + 3) % 4`로 처리한다.
- 청소한 칸은 2로 표시하여 벽(1)과 구분한다.

## 시간복잡도
- 방의 모든 칸을 최대 한 번씩 청소하고, 각 칸에서 4방향을 검사하므로 시간복잡도는 `O(N * M)`이다.

## 풀이 과정
1. 방의 크기 `N, M`, 로봇의 초기 위치 `r, c`와 방향 `d`, 방의 상태 `g`를 입력받는다.
2. 무한 루프를 시작한다.
3. 현재 칸(`g[r][c]`)이 청소되지 않았다면(`0`), 청소하고(`2`) `ans`를 증가시킨다.
4. 주변 4칸을 탐색하기 위한 플래그 `move = False`를 설정한다.
5. 4번 반복하며 왼쪽으로 회전(`(d+3)%4`)하고, 앞 칸이 청소되지 않은 빈 칸이면 이동하고 `move = True`로 설정 후 탐색을 중단한다.
6. 4방향 탐색 후 `move`가 `False`라면(이동할 곳이 없다면), 후진 방향(`(d+2)%4`)을 계산한다.
7. 후진할 칸이 벽(`1`)이면 루프를 탈출하고, 아니면 로봇을 후진시킨다.

## 코드 설명
```python
import sys
input = sys.stdin.readline

N, M = map(int, input().split())
r, c, d = map(int, input().split())
g = [list(map(int, input().split())) for _ in range(N)]
dr = [-1,0,1,0] # 북, 동, 남, 서
dc = [0,1,0,-1]
ans = 0
while True :
    if g[r][c] == 0 :
        g[r][c] = 2
        ans += 1
    move = False
    for _ in range(4) :
        d = (d+3)%4
        nr = r + dr[d]
        nc = c + dc[d]
        if 0 <= nr < N and 0 <= nc < M and g[nr][nc] == 0 :
            r = nr
            c = nc
            move = True
            break
    if not move :
        back = (d+2)%4
        nr = r + dr[back]
        nc = c + dc[back]
        if g[nr][nc] == 1:
            break
        else :
            r = nr
            c = nc

print(ans)
```
- 방향 벡터 `dr, dc`를 `[북, 동, 남, 서]` 순서로 정의하여 문제의 방향 인덱스와 일치시킨다.
- 반시계 회전과 후진은 모듈러 연산(`%4`)을 사용하여 간단하게 구현한다.
- `g[nr][nc] == 0` 조건을 통해 이미 청소된 칸(2)이나 벽(1)을 걸러낸다.
