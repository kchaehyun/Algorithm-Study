# Programmers 12927 - 야근 지수

## 문제 설명
- [문제 링크](https://school.programmers.co.kr/learn/courses/30/lessons/12927)
- 남은 작업량 배열 `works`와 작업할 수 있는 시간 `n`이 주어진다.
- 한 시간 동안 작업량 하나를 `1`만큼 줄일 수 있다.
- `n`시간 동안 작업한 뒤 남은 각 작업량의 제곱을 더한 값을 야근 지수라고 한다.
- 야근 지수의 최솟값을 반환한다.

## 아이디어
- 야근 지수는 남은 작업량의 제곱 합이므로, 작업량이 큰 일을 먼저 줄이는 것이 가장 효과적이다.
- 예를 들어 작업량 `a`와 `b`에 대해 `a >= b`라면, `a`를 1 줄였을 때 감소하는 야근 지수는 `2a - 1`이고 `b`를 1 줄였을 때는 `2b - 1`이다.
- 따라서 매 시간 현재 작업량이 가장 큰 일을 선택해 `1`씩 줄인다.
- 현재 최댓값을 빠르게 꺼내고 갱신하기 위해 최대 힙인 `priority_queue`를 사용한다.
- 모든 작업을 끝냈다면 남은 시간과 관계없이 야근 지수는 `0`이다.

## 시간복잡도
- 작업의 개수를 `M`이라고 할 때, 힙을 구성하는 데 O(M log M)이 걸린다.
- 최대 `n`번 작업하며 각 힙 연산은 O(log M)이므로 O(n log M)이 걸린다.
- 마지막으로 남은 작업량을 모두 꺼내 제곱 합을 구하는 데 O(M log M)이 걸린다.
- 전체 시간복잡도는 O((M + n) log M)이다.
- 힙에 모든 작업량을 저장하므로 공간복잡도는 O(M)이다.

## 풀이 과정
1. `works`의 모든 작업량을 최대 힙에 넣는다.
2. 최대 `n`번 반복하면서 힙의 최댓값을 꺼낸다.
3. 최댓값이 `0`이면 모든 작업이 끝난 것이므로 `0`을 반환한다.
4. 꺼낸 작업량을 `1` 감소시킨 뒤 다시 힙에 넣는다.
5. 작업 시간이 끝나면 힙에 남은 모든 작업량의 제곱을 더해 반환한다.

## 코드 설명
```cpp
long long solution(int n, vector<int> works) {
    long long answer = 0;
    priority_queue<int> pq;
    for(int w : works) pq.push(w);
    for(int i = n ; i > 0 ; --i) {
        int maxVal = pq.top();
        if(!maxVal) return 0;
        pq.pop();
        pq.push(--maxVal);
    }
    while(!pq.empty()) {
        int n = pq.top();
        pq.pop();
        answer += n*n;
    }
    return answer;
}
```
- `priority_queue<int>`는 가장 큰 작업량이 항상 `top()`에 위치하는 최대 힙이다.
- 매 반복마다 가장 큰 작업량을 꺼내 `1` 감소시킨 후 다시 넣는다.
- 힙의 최댓값이 `0`이면 다른 작업량도 모두 `0`이므로 즉시 `0`을 반환한다.
- 모든 작업을 마친 후 남은 작업량을 하나씩 꺼내 제곱한 값을 `answer`에 누적한다.
- 야근 지수의 합이 커질 수 있으므로 반환값과 `answer`는 `long long`을 사용한다.
