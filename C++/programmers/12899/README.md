# Programmers 12899 - 124 나라의 숫자

## 문제 설명
- [문제 링크](https://school.programmers.co.kr/learn/courses/30/lessons/12899)
- 124 나라는 자연수를 표현할 때 `1`, `2`, `4`만 사용한다.
- 자연수 `n`이 주어졌을 때, 이를 124 나라에서 사용하는 숫자로 변환해 반환한다.

## 아이디어
- 124 나라의 숫자는 3진법과 비슷하지만 숫자 `0` 대신 `4`를 사용한다.
- `n`을 3으로 나눈 나머지가 `1` 또는 `2`이면 해당 값을 결과에 추가하고 `n`을 3으로 나눈다.
- 나머지가 `0`이면 `4`를 추가하고, 자리 올림을 보정하기 위해 `n`을 `n / 3 - 1`로 갱신한다.
- 나머지는 낮은 자리부터 구해지므로 마지막에 문자열을 뒤집는다.

## 시간복잡도
- 숫자의 자릿수만큼 반복하므로 O(log N)이다.
- 결과 문자열을 뒤집는 과정도 O(log N)이므로 전체 시간복잡도는 O(log N)이다.

## 풀이 과정
1. 변환한 숫자를 저장할 문자열 `answer`를 만든다.
2. `n`이 0보다 큰 동안 `n`을 3으로 나눈 나머지를 확인한다.
3. 나머지가 0이면 `4`를 추가하고 `n`을 `n / 3 - 1`로 갱신한다.
4. 나머지가 1 또는 2이면 해당 숫자를 추가하고 `n`을 3으로 나눈다.
5. 만들어진 문자열을 뒤집어 높은 자리부터 정렬한 뒤 반환한다.

## 코드 설명
```cpp
string solution(int n) {
    string answer = "";
    while(n > 0) {
        if(!(n%3)) {
            answer += '4';
            n = n/3 - 1;
        }
        else {
            answer += to_string(n%3);
            n /= 3;
        }
    }
    reverse(answer.begin(), answer.end());
    return answer;
}
```
- `!(n % 3)`은 `n`이 3으로 나누어떨어지는지 확인한다.
- 나머지가 0일 때는 124 나라에 숫자 `0`이 없으므로 `4`를 사용한다.
- 이때 `n / 3 - 1`로 몫을 보정해야 앞자리까지 올바르게 변환된다.
- 각 자리의 숫자가 낮은 자리부터 추가되므로 `reverse()`로 문자열을 뒤집는다.
