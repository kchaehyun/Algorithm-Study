# Programmers 12938 - 최고의 집합

## 문제 설명
- [문제 링크](https://school.programmers.co.kr/learn/courses/30/lessons/12938)
- 자연수 `n`개로 이루어진 집합의 합이 `s`가 되도록 한다.
- 가능한 집합 중 각 원소의 곱이 최대가 되는 집합을 오름차순으로 반환한다.
- 조건을 만족하는 집합을 만들 수 없다면 `{-1}`을 반환한다.

## 아이디어
- 합이 일정할 때 곱을 최대로 만들려면 원소들의 차이가 최대한 작아야 한다.
- 예를 들어 두 수 `a`, `b`가 있고 `a + 2 <= b`라면, `a + 1`, `b - 1`로 바꿔도 합은 같지만 곱은 더 커진다.
- 따라서 모든 원소를 `s / n`으로 먼저 채운 뒤, 나머지 `s % n`만큼의 원소에 `1`씩 더한다.
- 오름차순으로 반환해야 하므로 `1`을 더하는 원소는 배열의 뒤쪽부터 배치한다.
- `n > s`이면 자연수 `n`개로 합 `s`를 만들 수 없으므로 `{-1}`을 반환한다.

## 시간복잡도
- 길이가 `n`인 배열을 만들고 나머지만큼 뒤쪽 원소를 갱신하므로 시간복잡도는 O(N)이다.
- 정답 배열을 저장하므로 공간복잡도는 O(N)이다.

## 풀이 과정
1. `n > s`이면 자연수 `n`개를 만들 수 없으므로 `{-1}`을 반환한다.
2. `s % n`을 구해 `1`을 더해야 하는 원소의 개수를 저장한다.
3. 길이가 `n`이고 모든 값이 `s / n`인 `answer`를 만든다.
4. 배열의 뒤쪽 `remainder`개 원소에 `1`씩 더한다.
5. 완성된 `answer`를 반환한다.

## 코드 설명
```cpp
vector<int> solution(int n, int s) {
    if(n > s) return {-1};
    int remainder = s%n;
    vector<int> answer(n, s/n);
    for(int i = n-1 ; i >= n-remainder ; --i)
        answer[i]++;
    return answer;
}
```
- `n > s`이면 최소 합인 `1 * n`보다 `s`가 작기 때문에 조건을 만족할 수 없다.
- `vector<int> answer(n, s / n)`으로 모든 원소를 가장 균등한 기본값으로 채운다.
- `remainder`는 기본값으로 채우고 남은 합이며, 이 개수만큼 원소를 `1` 증가시킨다.
- 뒤쪽 원소부터 증가시키면 배열이 오름차순 상태를 유지한다.
