# Programmers 12905 - 가장 큰 정사각형 찾기

## 문제 설명
- [문제 링크](https://school.programmers.co.kr/learn/courses/30/lessons/12905)
- `0`과 `1`로 이루어진 2차원 배열 `board`가 주어진다.
- 값이 `1`인 칸들로 이루어진 가장 큰 정사각형의 넓이를 반환한다.

## 아이디어
- 동적 계획법을 사용하여 각 칸을 오른쪽 아래 꼭짓점으로 하는 가장 큰 정사각형의 한 변 길이를 구한다.
- 현재 칸이 `1`이라면 위쪽, 왼쪽, 왼쪽 위 대각선 칸에 저장된 값 중 최솟값에 `1`을 더한다.
- 세 방향 중 하나라도 만들 수 있는 정사각형이 작으면 현재 칸에서 만들 수 있는 정사각형도 그 크기로 제한되기 때문이다.
- 첫 번째 행과 첫 번째 열은 이전 칸 세 개를 모두 확인할 수 없으므로 기존 값을 그대로 사용한다.
- 전체 칸에서 구한 가장 긴 변의 길이를 제곱해 넓이를 반환한다.

## 시간복잡도
- 행의 수를 `N`, 열의 수를 `M`이라고 할 때 모든 칸을 한 번씩 확인하므로 O(N * M)이다.
- 입력으로 복사된 `board`를 DP 배열로 사용하므로 별도의 추가 공간복잡도는 O(1)이다.

## 풀이 과정
1. 가장 긴 정사각형의 한 변 길이를 저장할 `maxLen`을 만든다.
2. `board`의 모든 칸을 순서대로 순회한다.
3. 현재 칸이 `0`이 아니고 첫 번째 행이나 열이 아니라면 위쪽, 왼쪽, 왼쪽 위 값의 최솟값에 `1`을 더해 현재 칸에 저장한다.
4. 현재 칸의 값으로 `maxLen`을 갱신한다.
5. 순회가 끝나면 `maxLen * maxLen`을 반환한다.

## 코드 설명
```cpp
int solution(vector<vector<int>> board)
{
    int maxLen = 0;
    
    for(int i = 0 ; i < board.size() ; ++i) {
        for(int j = 0 ; j < board[0].size() ; ++j) {
            if(board[i][j] != 0 && i > 0 && j > 0)
                board[i][j] = min({board[i-1][j], board[i][j-1], board[i-1][j-1]}) + 1;
            maxLen = max(maxLen, board[i][j]);
        }
    }

    return maxLen * maxLen;
}
```
- `board[i][j]`는 `(i, j)`를 오른쪽 아래 꼭짓점으로 하는 가장 큰 정사각형의 한 변 길이를 의미한다.
- `min({board[i-1][j], board[i][j-1], board[i-1][j-1]})`으로 위쪽, 왼쪽, 왼쪽 위에서 공통으로 확장할 수 있는 크기를 구한다.
- 현재 칸도 정사각형에 포함되므로 최솟값에 `1`을 더한다.
- 입력이 한 행 또는 한 열뿐인 경우에도 모든 칸에서 `maxLen`을 갱신하므로 올바르게 처리된다.
- 반환값은 한 변의 길이가 아니라 넓이이므로 `maxLen`을 제곱한다.
