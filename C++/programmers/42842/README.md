# Programmers 42842 - 카펫

## 문제 설명
- [문제 링크](https://school.programmers.co.kr/learn/courses/30/lessons/42842)
- 갈색 격자는 카펫의 테두리를 이루고, 노란색 격자는 그 안쪽을 채운다.
- 갈색 격자의 수 `brown`과 노란색 격자의 수 `yellow`가 주어졌을 때, 카펫의 가로와 세로 길이를 반환한다.
- 가로 길이는 세로 길이보다 같거나 길다.

## 아이디어
- 카펫의 전체 격자 수는 `brown + yellow`이다.
- 카펫의 가로와 세로는 전체 격자 수의 약수 쌍이다.
- 테두리를 제외한 안쪽 영역의 크기는 `(height - 2) * (width - 2)`이다.
- 이 값이 `yellow`와 같다면 조건을 만족하는 카펫이다.
- 세로 길이 `h`를 3부터 확인하면서 약수인 경우만 가로 길이 `w`를 계산한다.

## 시간복잡도
- 전체 격자 수를 `N`이라고 하면 세로 후보를 `sqrt(N)`까지만 확인하므로 O(sqrt(N))

## 풀이 과정
1. 전체 격자 수 `total`을 `brown + yellow`로 계산한다.
2. 세로 길이 후보 `h`를 3부터 `h * h <= total`인 동안 순회한다.
3. `total`이 `h`로 나누어떨어지지 않으면 카펫 크기가 될 수 없으므로 건너뛴다.
4. 나누어떨어진다면 가로 길이 `w`를 `total / h`로 계산한다.
5. 테두리를 제외한 내부 영역 `(h - 2) * (w - 2)`가 `yellow`와 같은지 확인한다.
6. 조건을 만족하면 `{w, h}`를 반환한다.
7. 조건을 만족하는 크기가 없다면 빈 벡터를 반환한다.

## 코드 설명
```cpp
vector<int> solution(int brown, int yellow) {
    int w = 0;
    int total = brown + yellow;
    for(int h = 3 ; h*h <= total ; ++h) {
        if(total%h != 0) continue;
        w = total / h;
        if((h-2)*(w-2) == yellow)
            return {w,h};
    }
    return {};
}
```
- `total`은 카펫 전체 칸의 수이다.
- `h`는 세로 길이 후보이고, `w`는 그에 대응하는 가로 길이이다.
- `h * h <= total`까지만 확인하면 가로가 세로보다 같거나 긴 약수 쌍만 검사할 수 있다.
- `(h - 2) * (w - 2)`는 갈색 테두리를 제외한 노란색 영역의 크기이다.
- 노란색 영역의 크기가 `yellow`와 같으면 정답 조건을 만족하므로 `{w, h}`를 반환한다.
