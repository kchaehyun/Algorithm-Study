# Programmers 12909 - 올바른 괄호

## 문제 설명
- [문제 링크](https://school.programmers.co.kr/learn/courses/30/lessons/12909)
- 문자열 `s`는 `(` 또는 `)`로만 이루어져 있다.
- 괄호가 올바르게 짝지어져 있으면 `true`, 그렇지 않으면 `false`를 반환한다.
- 올바른 괄호란 열린 괄호 `(`가 먼저 나오고, 그 뒤에 닫힌 괄호 `)`가 짝을 이루는 형태다.

## 알고리즘 유형
- 스택

## 아이디어
- 열린 괄호 `(`를 만나면 스택에 넣는다.
- 닫힌 괄호 `)`를 만나면 앞에 짝이 될 열린 괄호가 있어야 하므로 스택에서 하나를 꺼낸다.
- 닫힌 괄호를 만났는데 스택이 비어 있다면 짝이 맞지 않으므로 `false`를 반환한다.
- 모든 문자를 확인한 뒤 스택이 비어 있어야 모든 괄호가 올바르게 짝지어진 것이다.

## 시간복잡도
- 문자열의 길이를 `N`이라고 하면 O(N)

## 풀이 과정
1. 열린 괄호를 저장할 `stack<char>`를 만든다.
2. 문자열 `s`를 처음부터 끝까지 순회한다.
3. 현재 문자가 `(`이면 스택에 추가한다.
4. 현재 문자가 `)`이면 스택이 비어 있는지 확인한다.
5. 스택이 비어 있다면 짝이 될 열린 괄호가 없으므로 `false`를 반환한다.
6. 스택이 비어 있지 않다면 열린 괄호 하나와 짝을 맞춘 것이므로 `pop()`한다.
7. 순회가 끝난 뒤 스택이 비어 있으면 `true`, 남아 있으면 `false`를 반환한다.

## 코드 설명
```cpp
bool solution(string s)
{
    stack<char> sta;
    for(char c : s) {
        if (c == '(') sta.push(c);
        else {
            if (sta.empty()) return false;
            sta.pop();
        }
    }
    return sta.empty();
}
```
- `sta.push(c)`는 열린 괄호를 저장한다.
- 닫힌 괄호를 만났을 때 `sta.empty()`이면 닫을 수 있는 열린 괄호가 없으므로 올바르지 않다.
- `sta.pop()`은 가장 최근의 열린 괄호와 현재 닫힌 괄호를 짝지어 제거하는 의미다.
- 마지막에 `sta.empty()`를 반환해 짝지어지지 않은 열린 괄호가 남아 있는지 확인한다.
