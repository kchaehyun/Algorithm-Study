# Programmers 142085 - 디펜스 게임

## 문제 설명
- [문제 링크](https://school.programmers.co.kr/learn/courses/30/lessons/142085)
- 병사 `n`명과 무적권 `k`개를 가지고 연속된 적의 공격을 막아야 한다.
- `enemy[i]`는 `i + 1`번째 라운드에 등장하는 적의 수를 의미한다.
- 일반적으로 라운드의 적 수만큼 병사를 소모하며, 병사가 부족하면 게임이 종료된다.
- 무적권을 사용하면 병사를 소모하지 않고 해당 라운드를 통과할 수 있다.
- 무적권을 최대 `k`번 사용하여 통과할 수 있는 최대 라운드 수를 반환한다.

## 아이디어
- 무적권은 지금까지 등장한 적의 수가 가장 많은 라운드에 사용하는 것이 유리하다.
- 최소 힙에 각 라운드의 적 수를 넣고, 힙에는 무적권을 적용할 라운드만 남긴다.
- 힙의 크기가 `k`보다 커지면 가장 작은 적의 수를 꺼내 병사로 막는다.
- 이렇게 하면 힙에는 항상 지금까지 등장한 라운드 중 적이 가장 많은 `k`개가 남는다.
- 힙에서 꺼낸 적을 막을 병사가 부족하다면 현재 라운드에 도달하지 못하므로 해당 인덱스를 반환한다.

## 시간복잡도
- 라운드의 수를 `N`이라고 하면 각 적의 수는 힙에 한 번 삽입되고 최대 한 번 제거된다.
- 힙의 크기는 최대 `k + 1`이므로 전체 시간복잡도는 O(N log k)이다.
- 힙에 최대 `k + 1`개의 원소를 저장하므로 공간복잡도는 O(k)이다.

## 풀이 과정
1. 최소 힙에 첫 번째 라운드의 적 수를 넣는다.
2. 다음 라운드의 적 수를 최소 힙에 추가한다.
3. 힙의 크기가 `k`보다 크면 가장 작은 적의 수를 꺼낸다.
4. 꺼낸 적의 수만큼 병사를 사용한다.
5. 남은 병사로 적을 막을 수 없다면 현재 라운드 번호에 해당하는 인덱스를 반환한다.
6. 모든 라운드를 처리했다면 `enemy.size()`를 반환한다.

## 코드 설명
```cpp
#include <string>
#include <vector>
#include <queue>

using namespace std;

int solution(int n, int k, vector<int> enemy) {
    int i = 1;
    priority_queue<int, vector<int>, greater<int>> pq;
    
    pq.push(enemy[0]);
    while(!pq.empty() && i < enemy.size()) {
        pq.push(enemy[i]);
        if(pq.size() > k) {
            int minNum = pq.top();
            pq.pop();
            if(n < minNum) return i;
            n -= minNum;
        }
        ++i;
    }
    return enemy.size();
}
```
- `priority_queue<int, vector<int>, greater<int>>`는 가장 작은 값이 위에 위치하는 최소 힙이다.
- `pq`에는 무적권을 사용할 후보 라운드의 적 수가 저장된다.
- `pq.size() > k`이면 무적권으로 처리할 수 있는 라운드 수를 초과했으므로 가장 작은 값을 병사로 막는다.
- `n < minNum`이면 해당 적을 막을 병사가 부족하므로 현재까지 통과한 라운드 수인 `i`를 반환한다.
- 반복문이 끝날 때까지 병사가 부족하지 않았다면 모든 라운드를 통과한 것이므로 전체 라운드 수를 반환한다.
