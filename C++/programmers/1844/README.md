# Programmers 1844 - 게임 맵 최단거리

## 문제 설명
- [문제 링크](https://school.programmers.co.kr/learn/courses/30/lessons/1844)
- 게임 맵은 2차원 배열 `maps`로 주어진다.
- `1`은 이동할 수 있는 칸, `0`은 벽이라 이동할 수 없는 칸이다.
- 시작 위치는 왼쪽 위 `(0, 0)`이고, 상대 팀 진영은 오른쪽 아래 `(n - 1, m - 1)`이다.
- 상대 팀 진영까지 도착하는 최단 거리를 반환하고, 도착할 수 없으면 `-1`을 반환한다.

## 아이디어
- 가중치가 없는 격자에서 최단 거리를 구해야 하므로 BFS를 사용한다.
- 시작 지점부터 상하좌우로 이동 가능한 칸을 큐에 넣으며 탐색한다.
- BFS는 가까운 거리의 칸부터 방문하므로 처음 도착 지점에 도달했을 때의 거리가 최단 거리다.
- 별도의 거리 배열을 만들지 않고 `maps` 값을 현재 칸까지의 거리로 갱신한다.
- 방문한 칸은 `1`보다 큰 값으로 바뀌므로 다시 방문하지 않는다.

## 시간복잡도
- 모든 칸을 최대 한 번씩 방문하므로 O(N * M)

## 풀이 과정
1. 행의 개수 `R`, 열의 개수 `C`를 구한다.
2. 상하좌우 이동을 위한 방향 배열 `dr`, `dc`를 만든다.
3. BFS에 사용할 `queue<pair<int, int>>`를 만들고 시작 위치 `{0, 0}`을 넣는다.
4. 큐가 빌 때까지 현재 위치를 하나씩 꺼낸다.
5. 현재 위치가 도착 지점이면 `maps[r][c]`에 저장된 거리를 반환한다.
6. 상하좌우 인접 칸 중 범위 안에 있고 값이 `1`인 칸만 이동한다.
7. 이동할 칸의 값을 `현재 거리 + 1`로 갱신하고 큐에 넣는다.
8. BFS가 끝날 때까지 도착하지 못하면 `-1`을 반환한다.

## 코드 설명
```cpp
int solution(vector<vector<int> > maps)
{
    int answer = 0;
    int R = maps.size();
    int C = maps[0].size();
    int dr[4] = {1, 0, -1, 0};
    int dc[4] = {0, 1, 0, -1};
    queue<pair<int,int>> q;
    q.push({0,0});
    // maps[0][0] = 1;
    while(!q.empty()) {
        auto cur = q.front();
        q.pop();
        int r = cur.first;
        int c = cur.second;
        if(r == R-1 && c == C-1) return maps[r][c];
        for(int i = 0 ; i < 4 ; ++i) {
            int nr = r + dr[i];
            int nc = c + dc[i];
            if(0 <= nr && nr < R && 0 <= nc && nc < C && maps[nr][nc] == 1) {
                maps[nr][nc] = maps[r][c] + 1;
                q.push({nr,nc});
            }
        }
    }
    return -1;
}
```
- `dr`, `dc`는 아래, 오른쪽, 위, 왼쪽 방향으로 이동하기 위한 배열이다.
- `q`에는 다음에 탐색할 칸의 좌표가 저장된다.
- `maps[r][c]`는 시작 지점부터 현재 칸까지의 최단 거리로 사용된다.
- `maps[nr][nc] == 1`인 경우만 아직 방문하지 않은 이동 가능한 칸이다.
- `maps[nr][nc] = maps[r][c] + 1`로 다음 칸까지의 거리를 기록한다.
- 도착 지점에 도달하지 못하고 탐색이 끝나면 이동 경로가 없으므로 `-1`을 반환한다.
