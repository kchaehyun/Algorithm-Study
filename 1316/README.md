# BOJ 1316 - 그룹 단어 체커

## 문제 설명
- [문제 링크](https://www.acmicpc.net/problem/1316)
- 단어에 존재하는 모든 문자에 대해서, 각 문자가 연속해서 나타나는 경우만을 '그룹 단어'라고 한다.
- 예: `aba`는 `a`가 떨어져서 나타나므로 그룹 단어가 아니지만, `aabbbcc`는 그룹 단어이다.
- 주어진 `N`개의 단어 중 그룹 단어의 개수를 출력한다.

## 아이디어
- 단어를 한 글자씩 확인하며 이전에 나왔던 문자인지 체크한다.
- 이전에 나왔던 문자인데 바로 전 글자와 다르다면, 이는 떨어져서 다시 나타난 것이므로 그룹 단어가 아니다.
- 등장한 문자를 추적하기 위해 `set` 자료형을 사용한다.

## 시간복잡도
- 각 단어의 길이를 `L`이라고 할 때, 한 단어를 검사하는 데 `O(L)`이 소요된다.
- 전체 `N`개의 단어를 확인하므로 전체 시간복잡도는 `O(N * L)`이다.

## 풀이 과정
1. 단어의 개수 `N`을 입력받는다.
2. 각 단어를 순회하며 다음 과정을 수행한다.
3. 등장한 문자를 저장할 집합 `s`를 생성하고 첫 글자를 넣는다.
4. 두 번째 글자부터 마지막 글자까지 반복문을 돌린다.
5. 현재 문자가 `s`에 이미 존재하는데, 직전 문자(`word[i-1]`)와 다르다면 그룹 단어가 아니라고 판단(`p = False`)하고 중단한다.
6. 문자가 `s`에 없다면 `s`에 추가한다.
7. 그룹 단어인 경우(`p == True`) 카운트를 1 증가시킨다.

## 코드 설명
```python
import sys
input = sys.stdin.readline

N = int(input())
ans = 0

for _ in range(N) :
    word = input().strip()
    s = set()
    s.add(word[0])
    p = True
    for i in range(1,len(word)) :
        if word[i] in s :
            if word[i] != word[i-1] :
                p = False
                break
        else :
            s.add(word[i])
    if p :
        ans += 1
            
print(ans)
```
- `set`을 사용하여 이미 등장한 문자를 효율적으로 체크한다.
- `word[i] != word[i-1]` 조건을 통해 연속해서 나타나는 중인지, 아니면 떨어져서 다시 나타난 것인지 판별한다.
