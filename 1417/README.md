# BOJ 1417 - 국회의원 선거

## 문제 설명
- [문제 링크](https://www.acmicpc.net/problem/1417)
- 다솜이가 국회의원에 당선되기 위해 매수해야 하는 사람의 최솟값을 구하는 문제이다. 다솜이는 기호 1번이며, 다른 모든 후보의 득표수보다 많은 득표수를 가질 때 당선된다.

## 아이디어
### 방법 1: 그리디 (리스트 이용)
- 다솜이를 제외한 나머지 후보들의 득표수를 리스트에 저장한다.
- 매번 리스트에서 `max()` 값을 찾아 그 표를 1 빼고 다솜이에게 1을 더해준다.
- 다솜이의 표가 `max()`보다 커질 때까지 반복한다.

### 방법 2: 우선순위 큐 (최대 힙 이용) - **새로운 풀이**
- 나머지 후보들의 득표수를 최대 힙(Max-Heap)에 넣는다. (Python은 최소 힙만 지원하므로 음수로 변환하여 저장)
- 힙에서 가장 큰 값(`pop`)을 꺼내 다솜이보다 크거나 같으면, 1을 줄여 다시 힙에 넣고 다솜이의 표를 늘린다.
- 매번 리스트를 순회하여 최댓값을 찾는 방식보다 효율적이다.

## 시간복잡도
- 방법 1: 매 반복마다 `max()`를 찾으므로 `O(N)`이 걸려 전체적으로는 `O(매수한 사람 수 * N)`이다.
- 방법 2: 힙 연산(`push`, `pop`)은 `O(log N)`이므로 전체적으로 `O(매수한 사람 수 * log N)`이다.

## 풀이 과정 (우선순위 큐 기준)
1. 후보의 수 `N`과 다솜이의 초기 득표수를 입력받는다.
2. `N > 1`인 경우, 나머지 후보들의 득표수를 음수로 바꾸어 `heap`에 넣는다.
3. `while` 문을 통해 힙에서 꺼낸 가장 큰 값(절댓값 기준)이 다솜이의 표보다 크거나 같을 동안 반복한다.
4. 표를 하나 뺏어오고 `ans`를 증가시키며 힙을 갱신한다.
5. 반복이 끝나면 `ans`를 출력한다.

## 코드 설명
```python
import sys
import heapq
input = sys.stdin.readline

N = int(input())
dasom = int(input())
heap = []
ans = 0
if N > 1 :
    for _ in range(N-1) :
        v = int(input())
        heapq.heappush(heap, -v) # 최대 힙 구현을 위해 음수로 저장
    while True :
        max_val = heapq.heappop(heap)
        if -(max_val) >= dasom :
            max_val += 1 # 실제 값은 1 줄어드는 효과
            dasom += 1
            ans += 1
            heapq.heappush(heap, max_val)
        else : 
            break
print(ans)
```
- `heapq` 모듈을 사용하여 득표수가 가장 많은 후보를 효율적으로 관리한다.
- 다솜이의 표가 단독 최댓값이 될 때까지 표를 뺏어오는 그리디 전략을 사용한다.
